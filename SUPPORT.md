# Support

## Where To Ask Questions

- Use GitHub Discussions if discussions are enabled for this repository.
- Otherwise, open a GitHub issue for reproducible bugs or well-scoped feature requests.
- For local usage questions, container setup problems, or configuration help, start with [`README.md`](./README.md).

## What To Include

When asking for help, include:

- The version or image tag you are using.
- Whether you are using direct transport or proxy transport.
- Relevant runtime configuration, with secrets removed.
- Clear reproduction steps and any logs or screenshots that help explain the issue.

## Security Issues

Do not use public issues for security reports. Follow [`SECURITY.md`](./SECURITY.md).
